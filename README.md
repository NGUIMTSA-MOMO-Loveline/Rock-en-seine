# **Projet Tronc Commun**
## **🎵 Rock en Seine**
### **📌 Description**

Ce projet consiste à créer une page d’accueil pour le festival Rock en Seine dans le cadre d’un projet scolaire.
L’objectif est de proposer une interface moderne et attractive pour promouvoir le festival et inciter les utilisateurs à acheter des billets.

### **🎯 Objectifs**
- Présenter le festival Rock en Seine
- Mettre en avant les artistes et l’ambiance
- Inciter à l’achat de billets
- Créer une expérience utilisateur simple et fluide

### **🛠️ Technologies utilisées**
- HTML 
- CSS
